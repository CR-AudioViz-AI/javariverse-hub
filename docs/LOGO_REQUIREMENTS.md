# CR AudioViz AI Logo Requirements

## Logo Design Standards

All CR AudioViz AI app logos MUST follow these specifications for brand continuity:

### 1. 3D Style Guidelines

**Primary Style: Gradient + Shadow**
- Background: `linear-gradient(135deg, #0891b2 0%, #06b6d4 50%, #22d3ee 100%)`
- Shadow: `0 4px 14px 0 rgba(8, 145, 178, 0.39)`
- Border radius: Rounded corners (12px-16px for icons)

**Hover Effect:**
- Scale: 1.05x
- Shadow increase: `0 6px 20px 0 rgba(8, 145, 178, 0.5)`

### 2. Required Files Per App

| File | Size | Format | Usage |
|------|------|--------|-------|
| `favicon.ico` | 32x32 | ICO | Browser tab |
| `favicon.png` | 32x32 | PNG | Fallback |
| `icon-192.png` | 192x192 | PNG | PWA small |
| `icon-512.png` | 512x512 | PNG | PWA large |
| `apple-touch-icon.png` | 180x180 | PNG | iOS |
| `logo.svg` | Vector | SVG | Light mode |
| `logo-dark.svg` | Vector | SVG | Dark mode |
| `og-image.png` | 1200x630 | PNG | Social share |

### 3. Color Palette

**Primary (Cyan):**
- Main: `#0891b2` (cyan-600)
- Light: `#22d3ee` (cyan-400)
- Dark: `#0e7490` (cyan-700)

**Accent:**
- White: `#ffffff` for icons/text
- Slate-900: `#0f172a` for dark mode contrast

**FORBIDDEN in logos:**
- Purple, Violet, Green, Yellow, Orange, Pink, Indigo

### 4. Typography in Logos

- Font: Inter Bold or Semibold
- Color: White on gradient, Slate-900 on light backgrounds
- Minimum size: 24px for app name text

### 5. Icon Design Patterns

**App Category Icons:**

| Category | Icon Style | Example |
|----------|------------|---------|
| **Craft** | Yarn/Needle motif | Crochet, Knitting |
| **Collectors** | Display case/shelf | Cards, Coins, Vinyl |
| **Business** | Document/chart | Invoice, PDF, Resume |
| **Travel** | Globe/compass | Travel, Orlando |
| **Real Estate** | House/key | Realty, Property |
| **Media** | Play/film | Music, Video, Movie |
| **Games** | Controller/dice | Games, Arena |
| **Community** | Heart/hands | Veterans, Faith |
| **Health** | Heart/cross | Health, Fitness |

### 6. Cross-Branding Elements

All logos should include subtle CR AudioViz AI branding:
- Small "CR" badge in corner (optional)
- Consistent gradient direction (135°)
- Same shadow depth and color

### 7. Dark Mode Variants

- Invert text colors (white → slate-900)
- Reduce shadow opacity by 40%
- Keep gradient colors the same
- Increase contrast on borders
