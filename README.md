# CR AudioViz AI - Main Website

**Live at:** https://craudiovizai.com

## Deployment

This project is configured to auto-deploy to Vercel on push to main.

### First-Time Setup:
1. Go to https://vercel.com/new
2. Import this repository: CR-AudioViz-AI/javariverse-hub
3. Vercel will auto-detect Next.js and deploy
4. Add domains in project settings:
   - craudiovizai.com
   - www.craudiovizai.com

### Routes:
- `/` - Homepage
- `/chamber` - Multi-AI Chamber (ChatGPT + Claude + Javari)
- `/javari` - Javari AI Chat Interface
- `/apps` - Business Applications
- `/games` - Games Portal

## Tech Stack
- Next.js 14.2.35 (App Router)
- React 18
- TypeScript
- Supabase
- Tailwind CSS
