## Design System Overview

### 3D Style Requirements
All logos must follow this unified 3D aesthetic:

**Gradient:** `linear-gradient(135deg, #0891b2 0%, #06b6d4 50%, #22d3ee 100%)`
**Shadow:** `0 4px 14px 0 rgba(8, 145, 178, 0.39)`
**Border Radius:** 12-16px (rounded corners, not circles)
**Hover Effect:** Scale 1.05x with enhanced shadow

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Cyan-600 | #0891b2 | Primary gradient start |
| Cyan-500 | #06b6d4 | Gradient middle |
| Cyan-400 | #22d3ee | Gradient end/highlights |
| White | #FFFFFF | Icon/text on gradient |
| Slate-900 | #0f172a | Dark mode text |

### Required File Deliverables Per App
1. `favicon.ico` - 32x32 multi-size ICO
2. `favicon.png` - 32x32 PNG
3. `icon-192.png` - 192x192 PNG (PWA)
4. `icon-512.png` - 512x512 PNG (PWA)
5. `apple-touch-icon.png` - 180x180 PNG
6. `logo.svg` - Vector (light mode)
7. `logo-dark.svg` - Vector (dark mode)
8. `og-image.png` - 1200x630 PNG (social)

## Logo Creation Process

### Step 1: Design in Vector
- Use Figma, Illustrator, or similar
- Start with 512x512 artboard
- Apply 3D gradient background
- Add white icon/symbol
- Add shadow layer

### Step 2: Export Sizes
```
512x512 ‚Üí icon-512.png
192x192 ‚Üí icon-192.png (scaled)
180x180 ‚Üí apple-touch-icon.png
32x32 ‚Üí favicon.png
32x32 ‚Üí favicon.ico (convert)
```

### Step 3: Create Dark Mode Variant
- Keep gradient same
- Adjust icon contrast if needed
- Test on dark backgrounds

### Step 4: Create OG Image
- 1200x630 canvas
- Logo centered or left-aligned
- App name text
- Tagline (optional)

### Step 5: Deploy to Repository
Upload all files to `/public/` directory

## Cross-Branding Elements

To maintain ecosystem continuity:

1. **Consistent Gradient Direction** - Always 135¬∞ (top-left to bottom-right)
2. **Same Shadow Depth** - 4px blur, cyan-tinted shadow
3. **Rounded Corners** - 12-16px radius, never circular
4. **White Icons** - All icons/symbols in white on gradient
5. **Optional CR Badge** - Small "CR" in bottom corner for brand connection
